Module uorustlibs::hues [−] [src]

Methods for reading hue data out of hues.mul

Hues are represented in a continuous, unindexed file as groups - |header: u32|hues: [HueEntry..8]

Individual HueEntries are defined as |color_table:[u16..32]|table_start:u16|table_end:u16|name:[u8..20]|
